# AI Usage Report

## AI Tool Used
- ChatGPT

## How AI Was Used
I used ChatGPT in a limited and supportive way during this assignment. Most of the website structure, styling, and content were created and edited by me. AI was mainly used to help me improve some parts of the project and solve small implementation problems.

I used AI for:
- getting ideas for interactive features suitable for a portfolio website
- improving JavaScript logic for project filtering and form validation
- checking and refining some parts of the code structure
- getting help with organizing the required documentation files

## What I Did Myself
I built the main portfolio layout, updated the content, organized the sections, tested the interactive features, and adjusted the final design and code to match the assignment requirements. I reviewed all suggestions before applying them.

## Reflection
AI was useful as a support tool for learning and problem-solving, but I made the final decisions about the design, features, and final code included in the project.
