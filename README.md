# Assignment 2 - Interactive Portfolio Website

## Project Description
This project is an improved version of my personal portfolio website from Assignment 1. The goal of this version is to make the website more interactive and modern by adding JavaScript features, better user feedback, and smoother user experience.

The website includes dynamic behavior such as a theme toggle, time-based greeting, mobile navigation menu, live project search, and contact form validation.

## Features
- Dark/light theme toggle
- Theme preference saved using localStorage
- Greeting message based on time of day
- Responsive mobile navigation menu
- Live project search/filter
- Contact form validation with error and success messages
- Smooth transitions and hover effects
- Responsive layout for different screen sizes
